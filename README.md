# Pokédex

Simple React + TypeScript app that uses the public [PokéAPI](https://pokeapi.co/)
to display the first 50 Pokémon as a list of cards.

    Filtering & sorting:
- Filter by name
- Filter by type
- Sort A → Z / Z → A

Built with Vite, styled with Tailwind, icons from FontAwesome.

## Run it locally
Clone the repo locally and navigate to its directory.

    ```
npm install
npm run dev
```

Needs Node 18+ and npm. App runs on `http://localhost:5173`.
